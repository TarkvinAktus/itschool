### How do I set up ? ###

* Install webpack : npm install webpack -g
* Install Golang  : sudo apt-get install golang

### How do I start the application ? ###

* Build : webpack --progress --colors -p --watch
* Run   : go run server.go
* Navigate to http://localhost:3000/
